# 🏮 Lantern

A prototype [Data Catalogue](https://data.bas.ac.uk/-/index) for BAS discovery metadata.

## Overview

A data discovery tool used to find, evaluate and access resources produced, managed and/or used by the
[British Antarctic Survey](https://www.bas.ac.uk) and
[UK Polar Data Centre](https://www.bas.ac.uk/data/polar-data-centre/).

<!-- pyml disable md028 -->
> [!IMPORTANT]
> This project is temporary to formalise and build out refactored components initially developed in the
> [BAS Assets Tracking Service](https://github.com/antarctica/assets-tracking-service) that will form the next release
> of the [ADD Metadata Toolbox 🛡️](https://gitlab.data.bas.ac.uk/MAGIC/add-metadata-toolbox), which will form the new
> BAS Data Catalogue.

> [!NOTE]
> This project forms part of the MAGIC Spatial Data Infrastructure (SDI).

> [!NOTE]
> This project is focused on needs within the British Antarctic Survey. It has been open-sourced in case parts are of
> interest to others. Some resources, indicated with a '🛡' or '🔒' symbol, can only be accessed by BAS staff or
> project members respectively. Contact the [Project Maintainer](#project-maintainer) to request access.

## Project name

> What does Lantern mean?

Nothing, it's simply an unambiguous label to differentiate this project from other data catalogues. It was inspired by
the [MAGIC Lantern](https://www.magiclantern.fm/) Cannon camera project.

It was chosen to avoid generic terms and names that will get out of date, such as ‘new catalogue’ etc.
